Privacy Policy

Last updated: July 2026

1. What PowerGraph Is

PowerGraph is a fitness app for tracking workouts, calories, body weight, hydration, rankings, body fat estimates, and progress. The app can run locally in your browser, and backend sync is optional.

2. Local Browser Mode

When backend sync is not configured, PowerGraph stores your data in your browser using localStorage. This data stays on your device until you clear it in Settings, import different data, or clear your browser storage.

3. Optional Backend Sync

If backend sync is enabled, the backend may store account data, workouts, calorie and meal entries, body weight entries, body fat estimates, rest days, cheat days, app ratings, admin actions, and login events. Sync is used for backup, cross-device continuity, and admin review.

4. AI Features

AI food, ingredient, exercise, and body-related features may send the text or images you provide to the PowerGraph backend Gemini proxy. The frontend must never contain Gemini API keys, JWT secrets, or other private backend credentials.

5. Exports, Imports, and Deletion

  • You can export a JSON backup from Settings.
  • You can import a JSON backup back into the app.
  • You can clear local browser data from Settings.
  • If backend sync is used, deleting local data does not automatically delete server-side data.

6. Security Notes

API keys and JWT secrets must stay on the backend only. Do not put secrets into frontend VITE_* variables because frontend variables are visible in the browser bundle.

Politika zasebnosti

Zadnja posodobitev: julij 2026

1. Kaj je PowerGraph

PowerGraph je fitness aplikacija za sledenje treningom, kalorijam, telesni teži, hidraciji, rangom, ocenam telesne maščobe in napredku. Aplikacija lahko deluje lokalno v brskalniku, backend sinhronizacija pa je opcijska.

2. Lokalni način

Če backend sync ni nastavljen, PowerGraph shranjuje podatke v tvojem brskalniku prek localStorage. Podatki ostanejo na napravi, dokler jih ne izbrišeš v nastavitvah, uvoziš drugih podatkov ali počistiš podatkov brskalnika.

3. Opcijski backend sync

Če je backend sync omogočen, se lahko na backendu shranijo račun, treningi, kalorije in obroki, telesna teža, ocene telesne maščobe, rest/cheat dnevi, ocene aplikacije, admin akcije in dogodki prijave.

4. AI funkcije

AI funkcije za hrano, sestavine, vaje in telo lahko pošljejo besedilo ali slike, ki jih vneseš, na PowerGraph backend Gemini proxy. Gemini API ključ, JWT skrivnosti in drugi zasebni ključi ne smejo biti v frontend VITE_* spremenljivkah.

5. Izvoz, uvoz in izbris

  • V Settings lahko izvoziš JSON varnostno kopijo.
  • JSON varnostno kopijo lahko uvoziš nazaj v aplikacijo.
  • Lokalne podatke lahko izbrišeš v Settings.
  • Če uporabljaš backend sync, lokalni izbris ne izbriše nujno tudi podatkov na strežniku.

6. Varnost

API ključi in JWT skrivnosti morajo ostati samo na backendu. Skrivnosti ne dodajaj v frontend VITE_* spremenljivke, ker so vidne v brskalniku.
